Custom functionality plugin for G2F Design theme including custom post types and Gutenberg blocks.

== Description ==

G2F Functionality is a companion plugin for the G2F Design WordPress theme. It provides custom post types and Gutenberg blocks for showcasing projects, testimonials, and client logos.

= Features =

* **Projects Custom Post Type** - Showcase your portfolio with filterable categories
* **Testimonials Custom Post Type** - Display customer reviews and feedback
* **Clients Custom Post Type** - Show client logos in a marquee carousel
* **Project Grid Block** - Filterable grid display for projects
* **Testimonial Slider Block** - Carousel slider for testimonials
* **Client Marquee Block** - Animated logo carousel for clients

= Requirements =

* WordPress 6.4 or higher
* PHP 8.0 or higher
* G2F Design theme (recommended)

== Installation ==

1. Upload the `g2f-functionality` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start adding Projects, Testimonials, and Clients from the admin menu
4. Use the custom blocks in the Gutenberg editor

== Frequently Asked Questions ==

= Do I need the G2F Design theme? =

While the plugin works with any theme, it's designed to work seamlessly with the G2F Design theme for optimal styling.

= How do I add project categories? =

Navigate to Projects > Categories in the WordPress admin to add and manage project categories.

= Can I customize the block styles? =

Yes, you can add custom CSS to your theme or use the block editor's built-in styling options.
